Chimera

An interactive molecular graphics program from UCSF. Includes surface shading of volumes.

See www.cgl.ucsf.edu/chimera

To run it, add an alias to your .cshrc file, e.g.:

alias chimera '/us8/repository/chimera/bin/chimera'

Then you can start it up by typing 'chimera'. Here's an example from:

Tools menu --> Volumes --> Volume Viewer
