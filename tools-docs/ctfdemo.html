Contrast Transfer Function model - CTFDEMO

Usage: ctfdemo

Type "ctfdemo" at the Unix prompt, and experiment with the various CTF parameters used in SPIDER's TF commands.

Options

Grid : Turns a background grid on/off

Show envelope : Displays the envelope function (in green) in addition to the CTF

Squared : Makes the data positive

See also: WEB: CTF Parameters,

and the other CTF references at the Single Particle Reconstruction page.
