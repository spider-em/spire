View files on disk. Contents of text files are displayed in right hand window. SPIDER images can be displayed.

Directory tree

The tree of the current directory is appears in the left window.

Directories are opened and closed by clicking on the plus and minus nodes.

If the directory has a large number of files, it will have a node which will not open. The maximum allowable directory is specified under Options.

File viewer

File contents or other information may be displayed in the right hand window by double clicking on the file or folder icons.

Text files can be viewed in the right hand window by double clicking on their icons.

Double clicking on a directory folder icon displays a listing of the directory contents.

Double clicking on an image will cause the image to be displayed in Jweb or some other file viewer.

If the image cannot be displayed, xplor will attempt to display some file information, such as image size.

If the file or folder is too big, a warning will be displayed. Maximum limits are set under Options.

Options

Under the File menu, Options lets you set the maximum size for files and directories.

Note: directory size refers to the size returned by the Unix stat command, not the number of files.
